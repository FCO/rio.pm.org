Livros & Resenhas

Veja abaixo alguns dos principais livros de referência para a linguagem Perl, com atenção para as mini-resenhas, para saber qual o melhor livro para você de acordo com seu atual estágio de desenvolvimento. Em alguns dos links é possível ver trechos dos livros e até mesmo comprá-los online.

Fique atento também ao widget ao lado, que lista os últimos livros de Perl publicados pela O'Reilly!


Quer submeter uma resenha nova? Entre em contato pela lista!

Livros de Cabeceira:


Perl Cookbook 		Encontre um programador Perl, e você encontrará uma cópia do "Cookbook" por perto. Este fantástico livro traz uma lista detalhada de problemas, soluções e exemplos práticos para programadores. Excelente guia de referência, mas não deve ser utilizado por quem ainda não conhece Perl.
 

Perl Best Practices 		Muitos programam por instinto, com hábitos convenientes ou estilos adquiridos durante o aprendizado da linguagem. Em uma linguagem onde há muitas maneiras de se fazer cada coisa, é importante contar com um guia que mostra e demonstra porquê algumas maneiras são melhores do que outras, e garantem códigos legíveis e de fácil manutenção em projetos grandes e em equipe.
 

Outros Livros


Beginning Perl		 Embora voltado para o programador iniciante, este livro não te trata como um idiota, e programadores experientes (em outras linguagens) não terão problemas em ir direto para o que os interessa. O livro cobre muito material, dos princípios de programação até orientação a objetos e desenvolvimento para Web, enfatizando sempre as melhores práticas e código legível e de fácil manutenção. A primeira edição está totalmente disponível online e, embora cubra apenas o Perl 5.6, ainda é uma excelente referência para aprender a linguagem.
 

Perl: How to Program 		Guia completo para programação com Perl, este livro enfatiza o uso da Common Gateway Interface (CGI) com Perl para criar poderosos conteúdos dinâmicos na Web para aplicações de comércio eletrônico. Sua estrutura altamente didática inclui centenas de linhas de código e capturas de telas, dicas de cuidados e boas práticas para seu código além de diversos apontadores para referências externas na Web.
 

Programming Perl 		Conhecido como o "Livro do Camelo" (sim, dromedários são camelos =) e escrito por grandes expoentes da linguagem – inclusive o autor do Perl, Larry Wall – este livro rapidamente tornou-se e até hoje é reconhecido como a bíblia da linguagem. Mas não é apenas um livro para aprender Perl: sua abordagem única em relação à linguagem e à cultura de sua comunidade fazem deste livro leitura obrigatória e, ainda que cubra apenas a versão 5.6 da linguagem, permanece atual na maioria de seus aspectos e ajuda o programador a entender como as coisas funcionam por fora e por dentro.
 

Perl Medic: Transforming Legacy Code		 Mais uma excelente fonte para programadores, "Perl Medic" apresenta dicas e estratégias de como manter, otimizar, refatorar e escalar programas em Perl... quer você os tenha escrito ou não! Conheça boas práticas para ambientes enterprise e veja diversos exemplos e estudos de casos em um formato divertido de ler e fácil de aplicar em seu código do dia a dia. Um livro ideal para aqueles que programam por profissão e/ou em ambientes colaborativos, para ser usado complementarmente ao PBP.
 

Perl for Oracle DBAs 		Este livro traz uma introdução ao Perl e muitos utilitários prontos. Serve tanto para programadores Perl que implementam algum controle sobre o Banco de Dados, quanto para DBAs que precisam aprender um pouco mais da linguagem. Se você tiver oportunidade, verá que é um livro que traz ferramentas, em sua maioria desconhecidas que atendem desejos para vários outros bancos de dados!
 - mda 2007/09/23
 

Programming Web Services with Perl 		Escrito pelo próprio autor do SOAP::Lite, traz informação para usuários novatos e intermediários de Web Services. Se você está começando, ou encontrando dificuldades em implementar algo, verifique o que este livro lhe traz. SOAP, WSDL, REST, XML Schema, Transport, dados complexos, Client/Server... o que você precisa saber, e o que ainda nem imagina que vai precisar. Em um só projeto eu teria economizado 3 meses de depuração e muita dor de cabeça se tivesse em mãos as valiosas informações contidas neste livro.
- mda 2007/09/23
 

Writing Perl Modules for CPAN 		Ensinando design e implementação de módulos a partir do zero, este livro oferece a desenvolvedores Perl um detalhado guia sobre como utilizar e contribuir para a Comprehensive Perl Archive Network(CPAN)
